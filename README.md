# Azion Packages

Azion Packages for Edge Computing.

## Packages

### Cookies

The cookie package allows you to easily manage cookies in your `Javascript or Typescript` applications. Provides classes and functions to **retrieve**, **create** and **manipulate** cookies.

**For instructions for use and information see:**

🍪 [Azion Cookies Package](./packages/cookies/README.md)
